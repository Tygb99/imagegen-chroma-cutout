## SVG And GIF Takeaways

- Prefer SVG over PNG for the same flat, color-changeable design; the PNG guide says the same design file should be uploaded as only one of SVG or PNG.
- Route 3D, gradients, photo-like texture, and complex raster art to PNG, not SVG.
- Do not make a GIF from filmed/video footage when the request is for a GIF element. The GIF guide describes animated illustrations; video belongs to the separate MP4 route.
- Video upload is gated by portfolio review, so do not promise DesignHub-ready MP4 submission unless the user already has permission.
- For AI-assisted SVG/GIF work, record the prompt/source and mark AI-generated content during upload when applicable.
