## SVG/GIF 핵심 정리

- 같은 평면 디자인이 PNG와 SVG 모두 가능하면 색상 변경이 가능한 SVG를 우선한다. PNG 가이드는 동일 디자인 파일을 SVG와 PNG 중 하나로만 올리라고 안내한다.
- 3D, 그라데이션, 사진 같은 질감, 복잡한 raster art는 SVG가 아니라 PNG로 보낸다.
- GIF 요소 요청을 촬영 영상으로 만들지 않는다. GIF 가이드는 움직이는 일러스트를 설명하며, 촬영 영상은 별도의 MP4 동영상 경로다.
- 동영상 업로드는 포트폴리오 심사로 권한이 부여되어야 하므로, 사용자가 권한을 보유하지 않은 상태에서 DesignHub-ready MP4 제출을 약속하지 않는다.
- AI를 사용한 SVG/GIF 작업은 프롬프트와 source를 기록하고, 업로드 시 필요하면 생성형 AI 콘텐츠로 표시한다.
